# AI-Powered Editor Features

ScriptScribbler includes comprehensive AI-powered tools to enhance your screenwriting experience. The AI can help with editing, suggestions, analysis, proofreading, and more.

## Overview

The AI editor functionality allows writers to get intelligent assistance while writing their screenplays. All AI features are powered by Google's Gemini AI and are designed to enhance your creativity, not replace it.

## AI Features

### 1. AI Chat Assistant

Your personal writing assistant available directly in the Editor tab.

**How to Use:**
1. Click the **AI FAB** (floating action button) in the bottom-right corner
2. Select "Open AI Chat"
3. Start chatting with your AI assistant

**What You Can Ask:**
- "Fix the spelling errors in this scene"
- "Make the dialogue more natural"
- "Improve the action description"
- "Edit this to be clearer"
- "Suggest a better way to describe this character"
- "Help me with pacing in this scene"
- "What should happen next in my story?"

**Features:**
- Natural language commands
- Context-aware suggestions
- Script modifications
- Creative brainstorming
- Story analysis
- Character development ideas

### 2. Context Menu AI Editing

Quick AI editing options available via right-click.

**How to Use:**
1. Select any text in the editor
2. Right-click on the selected text
3. Choose from quick AI editing options:
   - **Fix Spelling & Grammar**
   - **Improve Clarity**
   - **Enhance Dialogue**
   - **Polish Action**

**What Happens:**
- AI analyzes the selected text
- Provides suggestions you can apply or dismiss
- Shows original text and improved version
- Explains why the change improves your script

**Best Use Cases:**
- Quick fixes for typos and grammar
- Improving specific dialogue exchanges
- Polishing action descriptions
- Enhancing clarity of specific scenes

### 3. Suggest Improvements

Get quick, actionable suggestions for your scenes.

**How to Use:**
1. Click the AI FAB button
2. Select "Suggest Improvements"
3. Review suggestions for the current scene or selection

**Suggestion Types:**
- Dialogue improvements
- Action description enhancements
- Pacing adjustments
- Character voice consistency
- Scene structure recommendations

**Features:**
- Context-aware analysis
- Specific, actionable feedback
- High/medium/low confidence ratings
- One-click application of suggestions

### 4. Deep Analysis

Receive a comprehensive breakdown of your screenplay.

**How to Use:**
1. Click the AI FAB button
2. Select "Deep Analysis"
3. Wait for the comprehensive report

**Analysis Includes:**
- **Plot Structure**: Acts, turning points, pacing
- **Character Development**: Arc analysis, consistency
- **Dialogue Assessment**: Natural flow, character voice
- **Scene Analysis**: Purpose, effectiveness, placement
- **Pacing Recommendations**: Scene length, rhythm
- **Constructive Feedback**: Specific improvements for each element

**Best Used For:**
- Major script reviews
- Identifying structural issues
- Character arc evaluation
- Overall quality assessment
- Pre-submission preparation

### 5. Proofread Script

Comprehensive error checking for your entire screenplay.

**How to Use:**
1. Click the AI FAB button
2. Select "Proofread Script"
3. Review the suggestions list

**Checks For:**
- **Spelling Errors**: Typos and misspellings
- **Grammar Mistakes**: Incorrect grammar usage
- **Formatting Issues**: Industry standard compliance
- **Continuity Problems**: Scene and character consistency
- **Punctuation**: Proper punctuation usage

**Suggestion Display:**
- Original text
- Corrected version
- Explanation of the issue
- Apply or dismiss each suggestion individually

**Features:**
- High accuracy (deterministic AI settings)
- Comprehensive coverage
- Non-destructive (you choose what to apply)
- Batch processing available

### 6. Generate Logline

Instantly create a compelling one-sentence summary of your screenplay.

**How to Use:**
1. Navigate to the **Logline** tab
2. Click "Generate Logline" or use AI FAB
3. Review and edit the generated logline

**What Makes a Good Logline:**
- One or two sentences
- Introduces protagonist
- Describes central conflict
- Hints at what's at stake
- Compelling and clear

**AI Capabilities:**
- Analyzes your entire script
- Identifies main character and conflict
- Creates multiple variations
- Provides industry-standard format

### 7. Generate Character Profiles

AI-powered character creation and development.

**How to Use:**
1. Go to the **Characters** tab
2. Select a character
3. Click "Generate Profile" or use AI tools

**Generated Elements:**
- Character backstory
- Personality traits
- Physical description
- Character motivations
- Character arc suggestions
- Relationship dynamics

**Features:**
- Based on character's dialogue and actions
- Consistent with script content
- Editable and refineable
- AI-generated portraits available

### 8. Generate Notes

Brainstorm ideas and create production notes with AI.

**How to Use:**
1. Navigate to the **Notes** tab
2. Click "Generate Notes" or use AI
3. Review and organize generated ideas

**Note Types:**
- Plot ideas

## Scene Management AI Features

### Add Scene Button

Quickly add new scenes with AI assistance.

**Features:**
- Click "Add Scene" below the editor
- New scene heading inserted: "INT. NEW LOCATION - DAY"
- AI can suggest scene settings based on context
- Continue writing without manual formatting

### End Scene Functionality

Mark scenes as complete and enable editing.

**How to Use:**
1. Write your scene content
2. Click "End Scene" at the bottom of the scene block
3. Button changes to "Scene ended" with "Edit Scene" option

**What Happens:**
- Scene marked as complete
- Visual feedback provided
- Edit Scene functionality enabled
- Scene metadata can be updated

### Edit Scene

Modify scene details after completion.

**How to Use:**
1. Click "End Scene" on a completed scene
2. Click "Edit Scene"
3. Modify scene details in the Scenes tab:
   - Scene setting
   - Scene description
   - Estimated time in minutes

## How AI Editing Works

### Process Flow

1. **User Action**: Select text and right-click, or ask AI via chat
2. **Request Analysis**: AI orchestrator analyzes the request
3. **Context Gathering**: Relevant screenplay blocks collected
4. **AI Processing**: Appropriate AI flow processes the request
5. **Suggestion Display**: Suggestions shown to the user
6. **User Choice**: Apply, modify, or dismiss suggestions

### Context Awareness

The AI uses context from:
- Surrounding screenplay blocks
- Character information
- Previous scenes
- Genre and tone
- Industry standards

### Confidence Levels

AI suggestions include confidence ratings:
- **High**: Very likely to improve the script
- **Medium**: Good suggestion, may need adjustment
- **Low**: Consider carefully before applying

## Best Practices

### For Using AI Features

1. **Be Specific**: "Fix spelling in this scene" is better than "fix this"
2. **Select Wisely**: Select specific text for targeted improvements
3. **Review Suggestions**: AI suggestions are helpful but not always perfect
4. **Maintain Your Voice**: Use AI as an assistant, not a replacement for creativity
5. **Context Matters**: Provide context to get better suggestions
6. **Iterative Process**: Use AI suggestions to refine, then refine again

## Technical Details

### AI Model

**Model:** Google's Gemini 2.5 Flash
**Provider:** Genkit

**Benefits:**
- Fast response times
- High-quality suggestions
- Context-aware editing
- Industry knowledge

### Temperature Settings

Different AI tasks use different "temperatures" for creativity:

- **Proofreading**: 0.1 (deterministic, focused on correctness)
- **Editing**: 0.3 (balanced between consistency and creativity)
- **Writing Assistance**: 0.7 (more creative for suggestions)

### Error Handling

All AI flows include:
- Graceful fallbacks if AI is unavailable
- User-friendly error messages
- Offline support (features disabled when offline)
- No data loss on errors

### Privacy & Data

**Important Information:**
- AI requests are sent to Google's Gemini API
- No screenplay data is permanently stored by the AI service
- All suggestions are processed in real-time
- You maintain full control over your content
- API key required (kept secure in environment variables)

### Performance Optimization

AI features are optimized for:
- Fast response times
- Minimal server load
- Efficient token usage
- Debounced requests (avoiding spam)

## Configuration

AI writing assistance can be configured:

```typescript
// Configuration options (for developers)
const { config, toggleEnabled, setMinConfidence } = useAiWritingAssist();

// Toggle AI on/off
toggleEnabled();

// Set minimum confidence threshold (0-1)
setMinConfidence(0.7); // Only show suggestions with 70%+ confidence
```

Configuration is persisted in localStorage.

## Troubleshooting AI Features

### AI Features Not Working

**Possible Causes:**
1. Missing `GEMINI_API_KEY` in `.env.local`
2. Invalid API key
3. Network connection issues
4. API rate limits reached

**Solutions:**
1. Check environment variables
2. Verify API key is correct
3. Check internet connection
4. Wait and retry if rate limited

### AI Suggestions Are Inaccurate

**Remember:**
- AI is a tool, not a replacement for your judgment
- Review all suggestions carefully
- Refine and adjust AI outputs
- Use specific prompts for better results
- Provide more context for complex requests

### AI is Too Slow

**Tips:**
- Check internet connection
- Reduce context size (select smaller portions)
- Use during off-peak hours
- Consider upgrading API plan
